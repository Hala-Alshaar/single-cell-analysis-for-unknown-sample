# single-cell-analysis-for-unknown-sample
 Single-Cell RNA-seq Analysis (Seurat + SingleR)

This repository contains a complete workflow for analyzing a human single-cell RNA sequencing (scRNA-seq) dataset using **Seurat** and **SingleR** in R.

The pipeline includes quality control, normalization, dimensionality reduction, clustering, marker gene identification, cell cycle analysis, and automated cell type annotation.
